# Disciplinas-Libertas
Repositório de materiais utilizados nas disciplinas ministradas na Libertas - Faculdades Integradas, organizados por curso, disciplina e semestre.

Aqui serão disponibilizados:
- Slides das aulas;
- Códigos base;
- Códigos desenvolvidos em aula;
- Materiais complementares.

Os conteúdos são atualizados **ao longo do semestre**, conforme o andamento das disciplinas.

---

## Organização do repositório

Cada disciplina possui uma pasta própria, seguindo o padrão:

### Exemplos:
- `SIS-DWI-2026-1` → Sistemas de Informação – Desenvolvimento Web I – 2026/1  
- `SIS-EDII-2026-1` → Sistemas de Informação – Estrutura de Dados II – 2026/1  

---

## Estrutura interna das disciplinas

Dentro de cada pasta de disciplina, os materiais costumam estar organizados da seguinte forma:

- **Plano da disciplina**
- **Slides**
- **Códigos base** (exemplos organizados para estudo ou início de atividades)
- **Código desenvolvido em aula** (conteúdo construído ao vivo, durante as aulas)
- **Materiais complementares**, quando aplicável

Essa organização visa facilitar o acesso, a revisão de conteúdo e o acompanhamento das aulas.

---

## Observações importantes

- Este repositório é apenas para **disponibilização de materiais**.
- As **entregas de atividades e trabalhos dos alunos** são realizadas, conforme orientações específicas de cada disciplina.
